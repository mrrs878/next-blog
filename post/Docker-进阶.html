打包优化

  • 缓存依赖

    1. 先 COPY 该命令的依赖文件，而不是所有文件
    2. 执行该命令

    如：

    FROM node

RUN rm -rf /app
RUN mkdir /app
WORKDIR /app

# 1. 先 COPY 该命令的依赖文件，而不是所有文件
COPY package.json yarn.lock ./
# 2. 执行该命令
RUN yarn install

COPY . .
RUN yarn build

EXPOSE 3000

CMD [ "node", "dist/main" ]

删除不用的镜像

  • 删除悬空的镜像， docker image prune -a -f
  • 删除悬空的容器，docker container prune -f
