前端架构&软实力-发布自己的npm包

2020-08-23 22:24:1244

发布前准备

  • 安装npm

  • 注册npm账号

  • package.json相关配置

{
    "name": "@mrrs878/blog",
    "version": "0.0.1",
    "private": false,
    "repository": "https://github.com/mrrs878/blog"
}
  • 更改仓库地址(之前更改过为淘宝镜像)
# 查看
npm config get registry

# 设置
npm config set registry https://registry.npmjs.org

发布


npm login

npm publish --access publish
